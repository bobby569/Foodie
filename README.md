## Setup

### If you are using yarn

Run the following command first

```
$ meteor npm i -g yarn
```

Then install the package. **Make Sure do this every time pulling from the master**

```
$ meteor npm install
```

or

```
$ meteor yarn install
```

## Run the app

```
$ meteor npm start
```

or

```
$ meteor yarn start
```
