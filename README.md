## Setup

### If you are using yarn

Run the following command first

```
$ meteor npm i -g yarn
```

Then install the package. **Make Sure you do this every time you pull from the master**

```
$ meteor npm install
```

or

```
$ meteor yarn install
```

## Run the app

```
$ meteor run
```
