## Foodie

### Links

* [Node & npm](https://nodejs.org/en/)
* [yarn](https://yarnpkg.com/en/)
* [Meteor](https://www.meteor.com/)
* [React](https://reactjs.org/)
* [Material-UI](http://www.material-ui.com/#/)

#### If you are using yarn

Run the follow command first

```
$ meteor npm i -g yarn
```

### Setup

```
$ meteor npm install
```

or

```
$ meteor yarn install
```

### Start the app

```
$ meteor run
```
